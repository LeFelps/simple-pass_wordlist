What is this password generator?
This is a simple wordlist generator for "birthday + name + special character" password combinations. Meaning that after you input someone's birthay and name (plus abbreviations), this tool will generate all possible combinations of "strong" passwords.

What is this for?
It can be used to bruteforce pentest your applications and accounts.

What is pentest?
Pentest (penetration test) is a cybersecurity exercise where you try to find vulnerabilities on your own system.
